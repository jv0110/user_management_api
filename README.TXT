User management API made with Nodejs, Express, JWT for user authentication and TDD/BDD with Chai and Mocha

1 - Download de repository
2 - Open on the terminal and type npm install to install all dependecies
3 - Type node index.js to run the server or npm run dev to run with nodemon if you have it installed

Route to get all the users:
localhost:3000/api/users

route to get an specific route by id:
localhost:3000/api/user/id
